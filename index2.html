Important FAQ

what is the document object model (dom)?

DOM is a programming interface for the web document.which can interact with the web page.

what is event delegation in the context of the dom,and why is it useful?

একটি ইভেন্ট লিসেনার দ্বারা একাধিক এলিমেন্টকে পরিচালনা করার পদ্ধতিকে ইভেন্ট ডেলিগেশন বলে। নতুন কোনো এলিমেন্ট যুক্ত হলেও সেটাকে পরিচালনা করা যাই এই পদ্ধতি দ্বারা।

how do you manipulate an element,s attribute and style using the dom?

বিভিন্ন ধরনের মেথড যেমন:(classlIst.add(),classlIst.remove(),document.getElementBy = id,class,tagName,.addEventlistener(),etc )এবং প্রপার্টি যেমনঃ (src,href,id,class etc)দ্বারা ম্যানিপুলেট করা হয়।

how do you select an element from the dom?

By using document.getElementBy = id,class,tagName and querySelector and querySelectorAll
